== Description ==

This plugin allow advertisers to integrate 2Performant's affiliate system with your WooCommerce shop.

It's tested and works with all payment methods found by default in WooCommerce as well as the following payment methods: euplatesc.ro and netopia.

If your phone number is on the site according to: https://support.2performant.com/hc/en-us/articles/224718568-Cum-ascund-numerele-de-telefon-de-pe-site-pentru-traficul-din-afiliere- 
you will need to hide it. This plugin will help you hide it using its CSS selectors.

== Installation ==

Please see [Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins) in the WordPress Codex.

After it's activated go to your WP dashboard > WooCommerce > Settings > Integration > 2Performant > add your campaign unique and confirm from 2Performant.
